## 👋 Olá! 

Sou um **Desenvolvedor Full Stack Pleno** apaixonado por engenharia de software, focado em construir sistemas escaláveis, resilientes e de alta performance. Tenho forte atuação no ecossistema **Java (Spring Boot / Quarkus)** no backend e **Angular** no frontend, unindo design de código limpo com entregas que geram valor real para o negócio.

Atuo na construção de soluções de ponta a ponta, prezando pela performance e manutenibilidade do código — desde a otimização de bancos de dados (Oracle, PostgreSQL) até o desenvolvimento de interfaces fluidas. Para expandir minha visão de engenharia, atualmente curso uma **Pós-graduação em Arquitetura e Desenvolvimento em Java**, aprofundando meus conhecimentos em microsserviços, cloud e sistemas distribuídos.

### 🚀 Foco e Especialidades

* **Arquitetura de Software:** Clean Architecture, SOLID, Design Patterns, CQRS e Event Sourcing.
* **Sistemas Distribuídos:** Microsserviços, APIs RESTful, gRPC, GraphQL e mensageria assíncrona (Apache Kafka, RabbitMQ).
* **Ecossistema Web:** Desenvolvimento orientado a componentes com Angular, controle de estado e construção de grids de dados complexos.
* **Cloud & DevOps:** Computação Serverless (Cloud Run, AWS Lambda), Docker e esteiras de deploy.
* **Performance:** Refatoração de queries (N+1), processamento em lote (bulk-fetching) e programação reativa (Spring WebFlux).
